## Summary
LP-facing guide to the Anduin platform (Jan 2026 cover date). Covers the complete investor journey from sign-up through form completion, e-signature, document upload, and accessing support. Primarily describes the FundSub subscription experience from the investor's perspective, with Anduin branding. CS uses this as a reference for investor support.

## Key Takeaways
- Login via OTP (one-time password) email; optional 2FA setup recommended immediately.
- Collaborators: investors can invite others to help fill forms; collaborators can view, fill, sign, and submit; have same access unless document-access customization is applied.
- Form logic is conditional — irrelevant questions are greyed out based on investor type answers.
- E-signature: type / draw / upload image; additional signers can be added with drag-and-drop field placement.
- Document upload for AML/KYC included in subscription flow; tax forms can be filled inline.
- Investors contact support via "Contact this fund" button → points of contact for the specific fund.

## Key LP Flows

**Sign-up:** Invite email → "Begin your subscription" → Accept Privacy Policy & T&C → Request OTP → Enter 6-digit code → Verify

**Login:** fundsub.anduin.app → enter email → password or OTP

**Profile:** Profile icon → Profile settings → Update personal info, password, 2FA

**Collaborators:** "Manage collaborators" → Invite → Enter names/emails → Send. Collaborators get same access to form (fill, sign, submit) unless access is restricted per-document.

**Form completion:** Conditional logic greys out irrelevant questions. Mandatory questions marked with red asterisk (*).

**E-signature:** "Add e-signature" → select docs → add signer emails → navigate to signature fields → choose type/draw/upload → Finish signing. Additional signers: drag-and-drop signature fields, assign to specific signer.

**Document upload:** "Upload AML/KYC & other documents" → Add document / Add form. Documents categorized by type; uncategorized go to "Other documents."

**Support:** Help → "Contact this fund" → view fund's points of contact.
