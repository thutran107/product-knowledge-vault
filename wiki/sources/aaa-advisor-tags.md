## Summary
Release note for the Advisor Tag feature, which lets fund managers post real-time status updates or context directly on FundSub instead of relying on separate email threads. Tags entered by GPs sync instantly to the advisor's dashboard in Advisor Advantage, giving advisors quick reference without offline status chasing.

## Key Takeaways
- **What it does**: a new "Advisor Tag" column appears on the FundSub dashboard for funds using advisor workflows. GPs assign custom tags to subscriptions created by advisor teams (e.g., "Custodian approve," "pending custodian approval").
- **Where advisors see it**: tags are visible to advisors under the "Fund's" tab in Advisor Advantage for quick reference.
- **How to enable**:
  - *New funds using the advisor workflow*: turn on the "Enable Advisor workflow" toggle in the Admin Portal — the column appears automatically.
  - *Existing funds already using the advisor workflow*: toggle "Enable Advisor workflow" off and on again to activate the column. No data is lost in the process.
