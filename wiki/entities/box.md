## Overview
A cloud content management and file sharing platform. Anduin integrates with Box to automatically transfer subscription and supporting documents from FundSub into customer-defined folder structures.

## Key Facts
- One Integration Hub connection: **Document Retrieval** — FundSub → Box.
- Self-configured: customers enter their own Box credentials; no credential sharing with Anduin.
- Supports folder customization by document type: main subscription doc, AML/KYC, tax, etc.
- Target persona: FundSub customers preferring self-service document management downstream.
- Setup flow mirrors the SFTP integration (Install → Name → Link funds → Configure → Validate).

## Role in This Knowledge Base
Primary example of the Document Retrieval integration pattern. Also the clearest example of self-service credential configuration.
